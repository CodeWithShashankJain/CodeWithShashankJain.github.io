Cascading Style Sheets is a style sheet language used for describing the presentation of a document written in a markup language such as HTML. CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript. Wikipedia Developed by: World Wide Web Consortium (W3C) Contained by: HTML Documents Container for: Style rules for HTML elements (tags) Initial release: December 17, 1996; 24 years ago Latest release: CSS 2.1: Level 2 Revision 1; (April 12, 2016; 5 years ago)
